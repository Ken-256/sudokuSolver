This is a Sudoku Solver,
it is made using pygame and solves the board using backtracking.

To use:
	-launch solverGUI.py

	-Every time you left click a box, its value will increase by 1. Every time you right click a box, its value will decrease by 1.

	-press delete/backspace to clear board

	-press enter when ready to solve

An error will pop up if the board is not valid
